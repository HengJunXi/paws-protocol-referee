      About

      P.A.W.S. Protocol is a tactical 1v1 / 2v2 card game by Atelier Rayfish.

      This app lets you play the game's umpire mode without needing an actual third person — it privately resolves each attack in the umpire's place, revealing only which card survives or is eliminated. You must own the original board game to play — this is only an assisting tool.
